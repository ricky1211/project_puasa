# 🌙 Rating Puasa Ramadan

Aplikasi web untuk menghitung dan memvisualisasikan rating ibadah puasa Ramadan.

## Cara Deploy ke Vercel

### Option 1 — Vercel CLI (paling cepat)
```bash
npm i -g vercel
vercel
```

### Option 2 — Drag & Drop
1. Buka https://vercel.com/new
2. Drag & drop folder ini langsung ke browser
3. Klik Deploy ✅

### Option 3 — GitHub
1. Push folder ini ke GitHub repo
2. Import repo di https://vercel.com/new
3. Klik Deploy ✅

## Fitur
- 🎯 Input jumlah hari puasa via slider
- 📊 Visual diagram radial progress + bar chart
- ⭐ Sistem grade A–S dengan rating bintang
- 🎁 Barcode THR otomatis jika rating >80%
- 🌙 Tema Ramadan malam bertabur bintang
